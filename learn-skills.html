Navigation in Outlier

Outlier has a sidebar that allows users to move between different sections:

  • Home: Overview of tasks, earnings, and learning progress.
  • Tasks: Available assignments for users to complete.
  • Completed Tasks: A section to review finished assignments.
  • Earnings: Track payments and past transactions.
  • Learn Skills: Educational resources to improve qualifications.

How to Work on Outlier

  1. Browse Available Tasks: Check the "Tasks" section for new assignments.
  2. Complete Assignments: Follow instructions carefully.
  3. Submit Work for Review: Tasks are evaluated before approval.
  4. Monitor Completed Tasks: Check status in the "Completed Tasks" section.
  5. Track Earnings: The "Earnings" page updates payments in real time.
  6. Enhance Your Skills: Learn new skills to qualify for more tasks.
